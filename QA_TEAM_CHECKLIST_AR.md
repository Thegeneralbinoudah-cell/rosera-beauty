# قائمة فحص الفريق (نسخة عربية)

استخدمي هذه القائمة قبل أي إطلاق.  
ضعي علامة على كل بند بعد التحقق.

## معلومات الاختبار

- [ ] التاريخ:
- [ ] اسم المختبِر/المختبِرة:
- [ ] البيئة (تجريبية/إنتاج):
- [ ] نسخة التطبيق / رقم الكوميت:

## أ) تسجيل الدخول والتنقل الأساسي

- [ ] يمكن للعميل تسجيل الدخول
- [ ] يمكن للمشرف/الأدمن تسجيل الدخول
- [ ] يمكن للمالك/المشرف التشغيلي تسجيل الدخول
- [ ] الصفحات الأساسية تفتح بدون شاشة بيضاء
- [ ] لا يوجد كسر واضح في الواجهة

## ب) المتجر والمنتجات

- [ ] صفحة المتجر تعرض المنتجات
- [ ] صفحة تفاصيل المنتج تفتح بشكل صحيح
- [ ] الإضافة إلى السلة تعمل
- [ ] صفحة إتمام الطلب تعمل

## ج) إدارة المنتجات والمزوّدين (الأدمن)

- [ ] الأدمن يستطيع إنشاء منتج من `/admin/products`
- [ ] الأدمن يستطيع تعديل تفاصيل/سعر المنتج
- [ ] الأدمن يستطيع إضافة مزوّد من `/admin/providers`
- [ ] الأدمن يستطيع ربط المنتج بالمزوّد (SKU/مخزون/تكلفة)

## د) عمليات الشحن

- [ ] صفحة الشحن `/admin/shipping` تعمل
- [ ] يمكن إضافة شركة شحن
- [ ] يمكن إنشاء شحنة لطلب
- [ ] رقم التتبع يُحفظ بشكل صحيح
- [ ] يمكن تحديث حالة الشحنة بالتسلسل الصحيح

## هـ) تتبع الطلبات للعميل

- [ ] العميل يستطيع فتح `/orders`
- [ ] رقم التتبع ظاهر للطلبات المشحونة
- [ ] رابط التتبع يفتح بشكل صحيح
- [ ] سجل التحديثات (Timeline) ظاهر

## و) الثقة والعمليات

- [ ] صفحة `/admin/trust-ops` تعمل
- [ ] يمكن إضافة حدث تشغيلي على الطلب
- [ ] يمكن تحديث درجة المخاطر وعلامات المخاطر
- [ ] بطاقات SLA والقرب من خرق SLA تعرض أرقامًا

## ز) الأمان وصلاحيات الأدوار

- [ ] العميل لا يستطيع دخول صفحات الأدمن
- [ ] غير الأدمن لا يستطيع تعديل عمليات الشحن/المزوّدين
- [ ] الأدمن يستطيع الوصول لكل صفحات العمليات

## ح) بوابة الإطلاق النهائية

- [ ] تم تنفيذ البناء بنجاح (`npm run build`)
- [ ] لا توجد مشكلة مانعة للإطلاق
- [ ] قرار الإطلاق: GO / NO-GO
